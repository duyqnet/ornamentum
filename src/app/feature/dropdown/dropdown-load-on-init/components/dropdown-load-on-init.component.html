DropDown Load Data On Init

  • Dropdown load data on init feature is used when users want to load data in the component initializing.
  • [loadDataOnInit]="true or false" => Enable/Disable getting data on init event. The default value for this config is true.
  • When this configuration is false, data won't be loaded as expected and users must provide their own way of data loading.
