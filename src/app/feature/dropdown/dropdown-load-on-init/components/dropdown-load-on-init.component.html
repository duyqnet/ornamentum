Load Data On Init

Dropdown load data on init feature can be uses to take over the control of data binding when on component initialization. If this configuration is set to false, data won't be loaded as expected and user must provide their own way of data loading mechanism by explicitly.

What You Will See

Basic Usage

For enable/disable the data load on init configuration, simply bind a specific boolean value into ng-dropdown. the default value for this configuration is true.
