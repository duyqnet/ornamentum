Dropdown Overview

Ornamentum dropdown is an simple dropdown with extended features that a conventional dropdown not supports. As same as the Data Table it was built for focusing on deliver a feature rich, highly configurable, UI framework agnostic, fully responsive and minimal peer dependency support dropdown on top of the Angular Framework.

  • lodash
  • rxjs
