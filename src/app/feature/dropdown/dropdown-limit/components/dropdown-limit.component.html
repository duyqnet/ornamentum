Data Limit

This property is useful when on request data from external data source. Limit will allow users to define a number of items to be fetch on a single load and when on user view all the fetched dropdown display items, again request a data set from data source recursively until all the data receives.

What You Will See

Basic Usage

For enable limit in dropdown, it is mandatory to set, Load On Scroll to true. if limit is not specified by the user, default value (30) will be set as the dropdown data limit.
