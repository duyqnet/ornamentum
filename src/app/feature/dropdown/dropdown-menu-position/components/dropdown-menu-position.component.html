Dropdown Menu Position

  • menuPosition will allows users to configure the dropdown menu position.
  • Available values for this configuration can be seen as follows.
    • [menuPosition]="MenuPosition.BOTTOM_RIGHT" | "MenuPosition.BOTTOM_LEFT" | "MenuPosition.TOP_LEFT" | "MenuPosition.TOP_RIGHT" => Note that, there are 4 positions users can configure to align the dropdown. The default value for this config is MenuPosition.BOTTOM_LEFT.
