DropDown Display Tracked By Attribute

  • Display Tracked By attribute lets users to configure the attribute which has the value to be displayed for the item shown in the dropdown.
  • To set that, users need to provide key name for displayTrackBy attribute.
  • According to following example above property is set as follows.
    • [displayTrackBy]="'value'" => This 'value' key contains the item value which needs to show for a dropdown item. Refer display-tracked-by-usage.data.ts file to see the data list used in here.
