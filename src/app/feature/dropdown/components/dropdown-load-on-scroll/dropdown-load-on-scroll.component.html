DropDown Load Data On Scroll

  • Dropdown load on scroll feature is used to load data dynamically to the dropdown on user scroll event.
  • When the data set is huge without getting whole lot of data at once to the client side, we can use this feature to get additional data only when user scrolled down to the last item.
  • What happens is user send another request and get all the available next data.
  • To enable this users need to configure following properties.
    • [loadOnScroll]="true or false" => Enable/Disable getting data on user scroll event. The default value for this config is false.
    • [onDataBind]="onDataBind" => Represents Dropdown data bind event callback. THat means the function which is provided via this will be called when it is needed to get next available data set. Refer example to see how this can be implemented.
  • Please make sure to check the available data set size before using this feature.
