Dropdown Row Selection

  • This help users to do the item selection in the dropdown.
  • There are several configurations are available when using item selection.
    • [setFirstOptionSelected]="true or false" => Enable/disable setting first option selected in dropdown. The default value for this config is false.
    • [showOptionSelectCheckbox]="true or false" => Enable/disable item selection check boxes. The default value for this config is false.
    • [selectedOption]="Option value to be selected" => Used to set one option as selected in the dropdown.
    • [selectedOptions]="Option values to be selected" => Used to set multiple options as selected in the dropdown.
    • [selectMode]="DropdownSelectMode.SINGLE" | "DropdownSelectMode.SINGLE_TOGGLE" | "DropdownSelectMode.MULTI" => Enables users to configure the dropdown item selection type. There are 3 item selection types are available in the dropdown. The default value for this config is DropdownSelectMode.SINGLE_TOGGLE. The available types are,
      • Single item selection: DropdownSelectMode.SINGLE
      • Single toggle item selection: DropdownSelectMode.SINGLE_TOGGLE
      • Multi item selection: DropdownSelectMode.MULTI
    • [showClearSelectionButton]="true or false" => Enable/disable selected option(s) clear icon. The default value for this config is false.
    • [showSelectedOptionRemoveButton]="true or false" => Enable/disable one particular selected option clear icon. The default value for this config is false.
    • [multiSelectOptionMaxWidth]="Maximum width to show selected option" => Used to set maximum width value for a particular selected value in the multi item selection mode. The default value for this config is 135px.
    • [selectTrackBy]="'Unique Attribute Key Name'" => This needs to specify when needs to identify selected item(s) uniquely. Please refer dropdown events section for more details.

Single Item Selection

  • Users are only allowed to select one item at a time.
  • Once the selection is made, it can only be changed by selecting other item. That means, only one item can be selected.

Single Toggle Item Selection

  • Users are only allowed to select one item at a time.
  • Can toggle the item selection.
  • But in this type, once the selection is made, it can be removed by clicking the item itself or by unchecking the checkbox(if check boxes enabled).

Multi Item Selection

  • Users can select multiple items at a time.
  • Can toggle the item selection.

Item Selection With All Other options

  • The following example shows how all the other options can be used with the row selection. Note that, for this example multi selection mode is used and for the other selection types also, these additional configurations can be configured in the same way.
  • These are the configured properties in here.
    • [showOptionSelectCheckbox]="true"
    • [selectedOptions]="selectedOptions" => Refer example to to see the provided value for selectedOptions. If user want to select one item, single object can be provided via selectedOption attribute.
    • [showClearSelectionButton]="true"
    • [showSelectedOptionRemoveButton]="true"
    • [multiSelectOptionMaxWidth]="200"
