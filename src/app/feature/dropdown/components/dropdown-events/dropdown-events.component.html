Dropdown Events

  • There are several events types are available in this dropdown as follows.
      • Triggers when initializing the dropdown.
      • Triggers when dropdown item select state changes. To identify selected items uniquely it is necessary to specify [selectTrackBy]" property. Therefore, a unique value key name should be set as the value for this attribute. According to this example it is 'key'. If user wants to trigger select change event in the init that can be achieved by setting true for the available triggerSelectChangeOnInit configuration as follows.
        • [triggerSelectChangeOnInit]="true"
      • This will trigger after completion of data binding.
  • Please refer following example to see, how this event can be used and what types of data is passed to these events.
