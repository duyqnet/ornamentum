DropDown Filtering

  • Filtering lets users to see the data which belongs to a specified criteria.
  • To Enable filtering, it is required to set the following property into true.
    • [filterable]="true"
  • Note that, The default value for filterable property is false. When there is no value or false value is assigned to the filterable property, filtering feature won't be enabled for the dropdown.

Additionally, the following configurations can be configured with filtering when enabling dropdown filterable option.

  • [filterDebounce]="true or false" => Used to enable/disable dropdown filter debounce. The default value for this config is true.
  • [filterDebounceTime]="100 (time in number)" => Used to specify the dropdown filter search debounce time. The default value for this config is 500.
  • [filterText]="'Filter Text Value'" => Used to set default filter text value for the dropdown. The default value for this config is empty.
