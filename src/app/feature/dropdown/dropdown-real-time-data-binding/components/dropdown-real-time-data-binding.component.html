Real-time Dropdown Data Binding

Sometimes, users might want to update the exiting data in the dropdown, due to the dynamic nature of different data sets such as weather data, analytics data, etc...

In such cases, you can use this real-time data binding feature & when data changes in real time those will automatically reflect to the original dropdown.
