Client-side Data Binding

Client side data binding allow users to bind data set to data table when those are available in the client side side.

Client side binding can be configure by providing any array of objects into items property or providing a data stream to the dataSource property.

In this data binding technique, data set will be fetched at once and item search filtering happens in the client side.

If the data stream is very large and can't perform filtering in the client side, Server Side Data Binding can be used.
