Client-side Dropdown Data Binding

Client-side data binding will allow users to bind a static data set into data table directly when expected data is available in the client side. not like in Server-side Data Binding and Realtime Data Binding we do not need a data source which listening for responses recursively.

Basic Usage

In client-side data binding can be done by binding a array of any value type into items on ng-dropdown tag.
