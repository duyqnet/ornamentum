Dropdown Data Binding

  • To show data we need to bind the data to the dropdown. There are several types of bindings are available, when binding data to the dropdown.
    • Client Side Data Binding
    • Server Side Data Biding
    • Real-time Data Biding
  • Depending on the requirement users can select the best data binding method.

Client Side Data Binding

In client side data binding, source data should be passed via the items attribute => [items]="data"
