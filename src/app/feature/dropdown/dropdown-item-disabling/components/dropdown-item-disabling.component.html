DropDown Item Disabling

  • Disabling lets users to disable the dropdown items when it is necessary.
  • To disable dropdown item(s), it is required to provide the boolean valued (to indicate whether an item needs to be disable or not) key name for disabledTrackBy attribute.
  • According to following example above property is set as follows.
    • [disabledTrackBy]="'disabled'" => This 'disabled' key contains boolean value which is used to control the disable state of an item.
