Data Table Limit

Limit will allows users to configure the number of items to be displayed in a particular page.

With limit when pagination is available

The specified value([limit]="30") will be applied as the limit.

Without limit when pagination is available

Default value(10) will be applied as the limit.

With limit when pagination is not available

The specified value([limit]="110") will be applied as the limit.

Without limit when pagination is not available

The total item count will be applied as the limit.
