Data Table Header Details

Showing header details along with the header menu will help users to get a better understanding about what kind of data is shown and allow to customize overall grid table by providing few of rich data table configurations.

What You Will See


Basic Usage

To show header in the data table users need to set the showHeader property to true. The default value for show header is false

{{demoSnippet.headerDetails}}

By enabling showHeader the header section will be enabled and there are few additional properties that can be configure after enabling the header section. those properties will be explain in-detail in next sections.

Useful Properties

There are few more properties that affects to the behaviour of table header. All the available properties can be applicable in global level of grid configuration.

Title

title: string = 'any appropriate name'
Default Value: ''

Title property will allow to define a name for the grid table. by simply binding a string value to the title property will be displays on header section. providing a meaningful name is mandatory to get better understanding about the showing data for the users from the first impression.

Show Refresh Button

showRefreshButton: boolean = true or false
Default Value: false

This configuration can be uses to enable/disable data refresh button. by enabling the refresh button users able to update the grid with latest data by clicking on refresh icon that appears in header details bar.

When dealing with Server Side Data Binding this property is very useful. In here, after a users click on refresh button it will trigger the particular event and ornamentum will re-send a request by invoking the respective service call and render the data with newly arrived response.

Also this property is useful in no data retrieval scenario. for more about no data please refer the No Records Template.

Show Column Selector

showColumnSelector: boolean = true or false
Default Value: false

This property is uses to show or hide the column selector button. by enabling the column selector property, users can select/unselect the available columns that need to be shown in grid table.

Selected columns will be displays with available or default column width.
