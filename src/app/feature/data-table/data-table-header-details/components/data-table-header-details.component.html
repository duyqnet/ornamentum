Data Table Header Details

  • Showing header details along with the header menu will helps users to get some understanding about what kind of data is shown.
  • To show header in the data table users needs to set the showHeader property in to true. The default value for this is false.
  • In the header, it shows title of the table, available columns and data refresh button. Those can be configured as follows.
    • [title]="true or false" => Specify the table data title. The default value for this config is ''.
    • [showRefreshButton]="true or false" => Show/hide refresh button. The default value for this config is false.
    • [showColumnSelector]="true or false" => Show/hide the button to trigger the column selection menu. The default value for this config is false.
