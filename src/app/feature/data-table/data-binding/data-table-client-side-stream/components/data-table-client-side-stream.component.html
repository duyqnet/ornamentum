Client Side Stream Data Binding

Using client-side real-time data binding feature users can update the existing data in the data table recursively to represent very dynamic data such as traffic, weather, analytics and etc without having a manual refresh.

In such cases you can use this client-side real-time data binding technique to binding data, when data changes in real time those changes will be reflect without data table refresh, and immediately effect to the data table as well. for more about data table refresh refer Show Refresh Button section.
