Data Table Overview

The Ornamentum full fledged data table is a component which is created with extensive configuration support for Angular. This comes with several built in features such as data binding, pagination, sorting, filtering, searching, etc.

Peer Dependencies

Ornamentum data table requires following dependencies to work.
  • @angular/common
  • @angular/core
  • @angular/forms
  • lodash-es
  • rxjs

Functionality

In the following it shows the available features & functionalities inside the Data Table.
  • Data binding
    • Client side data binding
    • Server side data binding
    • Real-time side data binding
  • Sorting
  • Filtering
  • Pagination
  • Row selection
  • Expandable rows
  • Row grouping
  • Column resizing
  • Templating
    • Cell Template
    • Loading Spinner Template
    • No Records Template
  • Translations
  • Data Persistence
  • UI Theming
  • Events
