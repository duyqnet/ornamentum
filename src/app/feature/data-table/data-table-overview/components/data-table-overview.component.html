Data Table Overview

Full fledged data table component with extensive configuration support.

Feature Overview

  • Client/Server data binding
  • Live/real-time data binding
  • Pagination
  • Multi/Single column sorting
  • Row grouping
  • Multi/Single value filtering
  • Row selection
  • Expandable rows
  • Column resizing
  • Cell/Header templating
  • Translations
