Real-time Data Binding

In some situations, you might want to update the existing data in the data table recursively, due to a dynamic nature of different data sets such as weather data, analytics data, etc. those type of data's are possible to change in a small time of period.

In such cases you can use this real time data binding technique to binding data, when data changes in real time those changes will be reflect without data table refresh, and immediately effect to the data table as well. for more about data table refresh refer Show Refresh Button section.
