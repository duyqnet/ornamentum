Data Table Data Binding

  • The main objective of data table is to show data in tabular format.
  • In order to show data we need to bind the data to the data table. There are several types of bindings are available, when binding data to the ornamentum data table.
    • Client Side Data Binding
    • Server Side Data Biding
    • Real-time Data Biding
  • Depending on the requirement users can select the best data binding method.

Client Side Data Binding

In client side data binding, source data should be passed via the items attribute => [items]="data"
