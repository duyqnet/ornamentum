Client-side Data Binding

Client side data binding allow users to bind static data set to data table when data is available in the client side. It can be configure by providing any array of objects into items property or providing a data stream to the dataSource property.

In this data binding technique, data set will be fetched at once and all the filtering, sorting, data pagination operations handle in the client side.

If the data stream is very large and can't perform data handling operations(such as filtering, searching, etc.) in client side, Server Side Data Binding can be use instead of client side data binding.
