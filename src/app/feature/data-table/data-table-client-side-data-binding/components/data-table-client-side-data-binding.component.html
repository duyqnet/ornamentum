Client-side Data Binding

Client side data binding allow users to bind static data set to data table when data is available in the client side.

Client side binding can be configure by providing any array of objects into items property or providing a data stream to the dataSource property.

In this data binding technique, data set will be fetched at once and all the filtering, sorting, data pagination things handled in the client side.
