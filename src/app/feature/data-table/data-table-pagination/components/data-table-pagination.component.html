Data Table Pagination

Most of the time it turns out that the amount of data to be displayed can’t fit on a single page, or the page becomes ridiculously long. In such cases, you might want to turn pagination on.

What you will see


Ornamentum will display some controls at the bottom so users can jump between pages and set how many rows they want to display per page using the limit text box

Basic Usage

To enable pagination simply set pageable to true

{{demoSnippet.pagination}}

By default the table doesn’t paginate it’s contents, even if properties like “limit” is set, more on that later.

Useful properties

A few other properties effect the way pagination behaves

Page

[page] = number

Setting this property lets you control which page the table starts off showing, notice how this table starts off on the second page

Limit

[limit] = number

Controls how many rows are shown per page, for more information on this property see the Data Limit page
