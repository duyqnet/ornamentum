Data Table Pagination

Most of the time it turns out that the amount of data to be displayed can’t fit on a single page, or the page becomes ridiculously long. In such cases, you might want to turn pagination on.

What you will see


The Pagination controls will be displayed on the bottom of the grid.User can set the number of rows displayed on the grid page and navigation controls.

Basic Usage

To enable pagination, simply set [pageable] to true

{{demoSnippet.pagination}}

By default the table doesn’t paginate it’s contents; even if properties like “limit” is set. More on that will be explained later.

Useful properties

There are a few properties which effect the behaviour of Pagination controls.

Page

[page] = number

The [page] property lets you control which page the table starts off showing, notice how this table starts off on the second page

Limit

[limit] = number

Controls how many rows are shown per page. Further information on this property is available in Data Limit page
