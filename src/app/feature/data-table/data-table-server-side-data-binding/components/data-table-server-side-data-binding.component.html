Server-side Data Binding

Server side data binding allow users to bind a data set that are not available in client side as a static data set but available in external data source with dynamic features. Now, server side data binding technique is available to handle and process data in this kind of scenarios.

Server side data binding allows users to get dynamic data stream from the server which matches to the filter or a particular search.

That means, unlike the client side data binding, whenever user performs a filtering, sorting, data pagination action a new api request will be initiated to fetch data from the server which matches to the particular action.
