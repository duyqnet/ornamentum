Data Table Row Selection

  • This help users to do the row selection in the data table.
  • There are several configurations are available when using row selection.
    • [rowSelectable]="true or false" => Enable/disable table row selection.
    • [showRowSelectCheckbox]="true or false" => Enable/disable table row selection check boxes.
    • [selectOnRowClick]="true or false" => Enable/disable table row selection on row click event.
    • [selectMode]="'single'" | "'single_toggle'" | "multi'" => Enables users to configure the dropdown row selection type. There are 3 row selection types are available in the data table. The default value for this config is "single". The available types are,
      • Single row selection: "single"
      • Single toggle row selection: "single_toggle"
      • Multi row selection: "multi"
    • [selectTrackBy]="'Unique_Value_Column_Name'" => This needs to specify when needs to identify selected row(s) uniquely. Please refer data table events section.

Single Row Selection

  • Users are only allowed to select one row at a time.
  • Once the selection is made, it can only be changed by selecting other row. That means, only one row item can be selected.

Single Toggle Row Selection

  • Users are only allowed to select one row at a time.
  • Can toggle the row selection.
  • But in this type, once the selection is made, it can be removed by clicking the row itself(if select on row click is enabled) or by unchecking the checkbox(if check boxes enabled).

Multi Row Selection

  • Users can select multiple rows at a time.
  • Can toggle the row selection.
