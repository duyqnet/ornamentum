Data Table Row Selection

Sometimes users may need to select specific set of data from the data table to run their own operations. In such cases, you might want to turn row selectable on.

What You Will See


Basic Usage

To enable row selectable, simply set rowSelectable to true and by default rowSelectable is false.

{{demoSnippet.row_selectable}}

In default the table doesn't allow you to select rows. But if you are going to select row(s), there are some additional configurations available to do the job easily. Those properties will be discuss on Useful Properties section.

Useful properties

As mentioned earlier, there are some additional configurations which helps users to control the row selection.

Row Selection Mode

selectMode: string ="single" | "single_toggle" | "multi"
Default Value: "single"

This control allows you to configure the dropdown row selection type. There are 3 types of row selections are available in the data table. The available types are,

  1. Single Row Selection: "single"

    2. Users are only allowed to select one row at a time. Once the selection is made, it can only be changed by selecting other row. That means, only one row item can be selected.

  2. Single Toggle Row Selection: "single_toggle"

    3. Users are only allowed to select one row at a time. Can toggle the row selection. But in this type, once the selection is made, it can be removed by clicking the row itself (if select on row click is enabled) or by unchecking the checkbox (if check boxes enabled).

  3. Multi Row Selection: "multi"

    4. Users can select multiple rows at a time. Can toggle the row selection.

Show Row Select Check Box

showRowSelectCheckbox: boolean = true or false
Default Value: false

The page property lets you to enable/disable table row selection check boxes as shown in the Row Selection Mode section's examples. Selection checkboxes are enabled in all those examples.

Select On Row Click

selectOnRowClick: boolean = true or false
Default Value: false

The page property lets you to enable/disable table row selection on row click event as shown in the Row Selection Mode section's examples. Select on row click is enabled in all those examples.

Select Track By

selectTrackBy: string = "Unique Value Column Name"
Default Value: "id"

This control helps users to identify selected row(s) uniquely. When selecting row(s), data table will emit the value of the column specified by the user. Please refer Data Table Events section for more details. All the examples shown in Row Selection Mode section has "id" as selectTrackBy property.
