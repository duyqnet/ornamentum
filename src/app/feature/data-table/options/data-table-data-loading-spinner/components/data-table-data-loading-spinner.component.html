Data Loading Spinner

In some cases, the loading of the data might take longer time than user expects. For such cases you can configure Data loading spinner to show data fetch operation is in progress. Once after the success data retrieval process it will automatically disappears and fetched data will be displayed on the grid.

Loading spinner is most useful if you used Server Side Basic or Server Side Web Socket Data Binding as your data retrieval technique, because we can't guaranteed about how long will take to complete the data fetch operation.

What You Will See

Basic Usage

showLoadingSpinner: boolean   Default Value: true Target Component: <ng-data-table>

Loading spinner can be enabled by binding true to showLoadingSpinner property and by default Loading spinner is enabled.

Furthermore, you can customize data loading spinner by binding a custom template as you prefer. For refer the Loading Spinner Template section.
