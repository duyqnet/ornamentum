Substitute Rows

Substitute rows are useful to show empty rows if there is less amount of data is available than the declared limit. In such cases, ornamentum will leave the rest of the data rows as empty instead of showing an empty area. However, substitute rows will not displays if the data set is equally divided by limit whether the substitute rows are enabled.
