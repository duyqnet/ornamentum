Header Details

Showing header details will help users to get a better understanding about what kind of data is shown and allow to configure set of properties which useful for enhance usability and experience of data table.

What You Will See

Basic Usage

showHeader: boolean   Default Value: false Target Component: <ng-data-table>

To configure header, set the showHeader property to true. The default value for show header is false

Useful Properties

By enabling showHeader the header section facilitates few additional properties that can be configure.

Title

title: string  
Default Value: ''
Target Component: <ng-data-table> Depends On: [showHeader]

Title property will allow to define a name for the grid table. Bind a string to the title property will be displays on header section. We recommend to provide a meaningful name to get better understanding about the content of showing to the users from the first impression.

Show Refresh Button

showRefreshButton: boolean  
Default Value: false
Target Component: <ng-data-table> Depends On: [showHeader]

This configuration can be uses to show/hide data refresh button. by enabling the refresh button users can perform data refresh operation by their self.

When dealing with Server Side Basic or Server Side Web Socket Data Binding this property is very useful. In this example, After invoking refresh button ornamentum will re-fetch data from the source and load them after resetting table state or displaying No Records Template on top of data table body when on a data retrieval failed scenario.

Show Column Selector

showColumnSelector: boolean  
Default Value: false
Target Component: <ng-data-table> Depends On: [showHeader]

This property is uses to show/hide the column selector button. by enabling the column selector property, ornamentum displays all the Show In Column Selector enabled properties.
