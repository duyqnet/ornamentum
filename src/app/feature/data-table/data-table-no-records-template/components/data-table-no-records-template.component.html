No Records Template

No Records Template, facilitate the users to customize the table body by providing their own template to show when the data is not available, by providing respective mandatory fields for no records template end-user can be notify in a preferred manner in no data scenarios.

No records template can be integrated into expandable rows. For about row expanding template configurations please refer Row Expand Template section.

What You Will See

The above demo snippet will demonstrate a sample no data scenario. until a success data response, we have enabled Data Loading Spinner for purpose of visible the process of data retrieval and make the data table more attractive.

Basic Usage

Similar to the other templates, it is possible to bind a specific property #ngDataTableNoRecords into ng-template to indicate that there is no data available to be retrieved on data fetch call.

Apart from the predefined messages users are facilitated with to define own custom template as preferred if no data available to be retrieved on data fetch call.
