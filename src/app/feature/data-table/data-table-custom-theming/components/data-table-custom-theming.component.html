Custom Theming

The following SASS variables will describe their each responsibility and by overriding those variables, can be generate a new data table with different experience to match your preferences.

Variable Default Value Description
$data-table-icon-color #333 Icon color used in data table ( Row Expand, Refresh Table, Column Selector Menu, Table Header Sorting, Pagination )
$data-table-icon-close-color #d32f2f Icon color of remove search criteria in table header
$data-table-loading-animation-color #333 Data table Loading spinner icon color
$delete-color #d32f2f Border color of the invalid data input in pagination input boxes
$data-table-container-border .07rem solid #ddd Border width, style and color of the data table outer-edge
$data-table-container-border-radius-value 0 Specify the radius value of the data table outer-edge
$data-table-container-box-shadow-value none Box shadow value of the data table
