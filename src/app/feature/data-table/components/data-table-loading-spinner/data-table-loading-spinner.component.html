Data Table Loading Spinner

The feature enables showing a loading spinner when loading data into the data table. To see spinner it is necessary to enable the following configuration.

[showLoadingSpinner]="true or false"

If this enabled a spinner will be displayed until the completion of the table data loading.


Please note that this loading spinner functionality can be overridden by any external party by providing a custom spinner template. Refer Loading Spinner Template under Data Table Templates section for more details.
