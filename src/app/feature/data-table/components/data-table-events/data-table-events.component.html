Data Table Events

  • There are several events types are available in this data table as follows.
    • Triggers when initializing the data table.
    • For each table row this will trigger when binding row data.
    • Triggers when row select state changes.
    • For each table row cell this will trigger when binding cell data.
    • Triggers when clicks on table cell.
    • Triggers for each column before all the other events to set column configurations.
    • triggers when clicking table column header row.
    • Triggers when user clicks on table row.
    • Triggers when user double clicks on table row.
  • Please refer following example to see, how this event can be used and what types of data is passed to these events.
