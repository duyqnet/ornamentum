Data Table Filtering

  • Filtering lets users to see the data which belongs to a specified criteria.
  • There are multiple list of configurations are available when working with data table filtering.
  • To Enable filtering, it is required to set the following property into true in the required columns.
    • [filterable]="true"
  • Note that, The default value for filterable property is false. When there is no value or false value is assigned to the filterable property, both dropdown & text box filtering will be off for the specified columns.

Filter Dropdown

The following content shows the available configurations when using column dropdown filter.

  • [showDropdownFilter]="true or false" => Enable/disable table column dropdown filter. The default value for this config is false.
  • [dropdownFilterMenuPosition]="MenuPosition.BOTTOM_RIGHT" | "MenuPosition.BOTTOM_LEFT" | "MenuPosition.TOP_LEFT" | "MenuPosition.TOP_RIGHT" => Enables users to configure the dropdown filter position. Note that, there are 4 positions users can configure to align the dropdown. The default value for this config is MenuPosition.BOTTOM_LEFT. According to the following example,
    • Station Name column filter has the position of BOTTOM_RIGHT
    • Status Value column filter has the position of BOTTOM_LEFT
    • Available Bikes column filter has the position of TOP_LEFT
    • Address column filter has the position of TOP_RIGHT
  • [dropDownFilterShowOptionSelectCheckbox]="true or false" => Show/hide dropdown filter's option selection check boxes. If value is not specified check boxes won't be shown.
  • [dropdownFilterCloseMenuOnSelect]="true or false" => Enable/disable dropdown filter close after an option is selected. The default value for this config is true. According to the following example,
    • Station Name column filter has false value for this config
    • Station Name column filter has true value for this config
  • [dropdownFilterShowSelectAll]="true or false" => Show/hide dropdown filter all option selection check box. If value is not specified select all check box won't be shown.
  • [dropdownFilterSelectMode]="DropdownSelectMode.MULTI" | DropdownSelectMode.SINGLE" | DropdownSelectMode.SINGLE_TOGGLE" => Enables users to configure the dropdown filter option selection type. The default value for this config is DropdownSelectMode.MULTI. There are 3 types of selection types available in ornamentum data table dropdown filter.
    • MULTI mode: Enable multiple option selection
    • SINGLE mode: Enable single option selection
    • SINGLE_TOGGLE mode: Enable single option selection with toggle
  • [dropdownFilterWrapDisplaySelectLimit]="5 (limit in number)" => Used to configure the number of options to be shown in the dropdown at a time. The default value for this config is 1.
  • [dropdownFilterShowSelectedOptionRemoveButton]="true or false" => Used to show/hide the filtered option removal from the dropdown. The default value for this config is false.
  • [dropdownFilterGroupByField]="group field name" => Enable dropdown filter options grouping under a specified name. If the group name is not specified all the values are shown as a single group.
  • [dropdownFilterMenuHeight]="200 (height in number)" => Used to specify the column dropdown filter menu height. The default value for this config is 250.
  • [dropdownFilterMenuWidth]="200 (width in number)" => Used to specify the column dropdown filter menu width. The default value for this config is 320.
  • [dropdownFilterMultiSelectOptionMaxWidth]="300 (max width in number)" => Used to specify the multiple option selection dropdown option max width. The default value for this config is 135.
  • [dropdownFilterSearchable]="true or false" => Used to show/hide dropdown filter search box. The default value for this config is true.
  • [dropdownFilterSearchDebounce]="true or false" => Used to enable/disable dropdown filter debounce. The default value for this config is true.
  • [dropdownFilterSearchDebounceTime]="100 (time in number)" => Used to specify the dropdown filter search debounce time. The default value for this config is 500.
  • [dropdownFilterShowClearSelectionButton]="true or false" => Used to show/hide drop down filter selection clear button. The default value for this config is true.
  • [filter]="Column value filter" => Used to provide a filter to see the values matched for specified criteria. In here [{{ '{' }} key: "In Service", value: "In Service" {{ '}' }}] filter is applied for the Status Value column.

Filter Text Box

The following content shows the available configurations when using column search filter text box.

  • [filterPlaceholder]="Filter place holder text to display" => Used to specify placeholder name for search text box. If value is npt specified, filter text value is shown as empty.
  • [showFilterClearButton]="true or false" => Used to show/hide text box search filter clear button. If default value for this config is true.
