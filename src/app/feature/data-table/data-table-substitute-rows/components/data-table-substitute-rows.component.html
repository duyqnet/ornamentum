Data Table Substitute Rows

Sometimes you might want to show empty rows in the data table when there are less amount of data is available than the specified limit. Therefore, in such cases, you might need to turn the showSubstituteRows on.

What You Will See


Basic Usage

To enable showSubstituteRows, simply set showSubstituteRows to true and by default showSubstituteRows is false.

Without Substitute Rows

Substitute rows will not be shown although data limit is less than the specified limit, if showSubstituteRows property is set to false.
