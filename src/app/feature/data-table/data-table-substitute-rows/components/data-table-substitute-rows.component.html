Substitute Rows

Substitute rows are useful when you want to show empty rows in data table, if there are less amount of data is available than the specified limit. In such cases, you might need to turn showSubstituteRows enable to show empty rows for the display defined data table limit.

What You Will See

Basic Usage

To enable substitute rows, simply set showSubstituteRows to true and by default the property is in false state. In What You Will See section example limit is specified as 15, but it is showing 5 additional rows due to unavailability of last 5 data items.

Without Substitute Rows

Substitute rows will not be shown although data limit is less than the specified limit, if showSubstituteRows property is set to false. In the following example limit is specified as 15, but it is not showing 5 additional rows because showing substitute rows is not defined and therefore property was in default configuration state (false) or else can be explicitly define as false in table level configuration.
