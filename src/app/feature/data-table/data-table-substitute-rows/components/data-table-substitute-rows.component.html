Substitute Rows

Sometimes you might want to show empty rows in the data table when there are less amount of data is available than the specified limit. Therefore, in such cases, you might need to turn the showSubstituteRows on.

What You Will See

Basic Usage

To enable showSubstituteRows, simply set showSubstituteRows to true and by default the property is false.

In default the table doesn't show substitute rows although data limit is less than the specified limit. In the above example limit is specified as 15. But it is showing 5 additional rows due to unavailability of 5 data items.
