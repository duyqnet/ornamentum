Themes

Ornamentum data table now comes with different themes that you can use to style your own application. currently it's supports 05 different themes as shown in the below.

Without using the available data table themes you can create your own theme by overriding default data table configurations. for about theme customizations, refer Custom Theming section.

Including Theme Support

Currently ornamentum has different CSS/SCSS files to represent the available themes. Therefore, in order to include these themes into your project you have to import its CSS/SCSS source references to your style files.

Also in the page header you can switch overall user experience of available themes by clicking on the given color palette.

Base Theme

The following image shows how Ornamentum Data Table looks with the Base theme. This is the user experience that you and your end-users feel, if you choose to stay with the base themw without using any of the themes available.
