Data Table Column Sorting

This feature enables users to sort the column data according to their requirement. Following configurations are available when dealing with data table column sorting.

  • [sortable]="true or false" => Enables/disables column sorting configuration. The default value for this config is false.
  • [sortOrder]="SortOrder.ASC" | "SortOrder.DESC" | "SortOrder.NONE" => There are 3 types of sorting behaviours are available. If sorting method is not mentioned by the user, then the default sorting mechanism(SortOrder.NONE) will be applied for those columns. According to this example this is how sorting is applied.
    • ID column: SortOrder.ASC
    • Station Name & Status Value columns: SortOrder.ASC
    • Available Bikes & Address columns: SortOrder.NONE
