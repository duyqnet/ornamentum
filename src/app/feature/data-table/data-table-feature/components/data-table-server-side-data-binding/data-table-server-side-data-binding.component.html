Data Table Data Binding

  • The main objective of data table is to show data in tabular format.
  • In order to show data we need to bind the data to the data table. There are two types of bindings can be used, when binding data to the ornamentum data table.
    • Static Data Binding
    • Dynamic Data Biding
  • Depending on the requirement users can select the best data binding method.

Server Side Data Biding

In server side data binding, data should be passed via the dataSource attribute => [dataSource]="source"
