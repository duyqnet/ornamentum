Data Table Responsive Configuration

The following configuration are available when setting data table responsiveness.

[minWidth]="650", [minHeight]="200", [contentHeight]="400", [width]="800"
  • Specifying minWidth & minHeight will enable horizontal & vertical scroll bars if the container width and height met with the specified values.
  • width configuration is used to specify the table width.
  • contentHeight configuration is used to specify the table content height.

[minWidth]="650", [minHeight]="200"

  • if width is not specified data table will take the parent container width.
  • if contentHeight is not specified data table will take the parent container height.
