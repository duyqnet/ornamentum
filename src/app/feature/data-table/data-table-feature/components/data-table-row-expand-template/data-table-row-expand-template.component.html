Data Table Row Expand Template

This feature enables users to customize the table row expandable section by providing their own template. There are set of configurations needs to be enabled to see and control the behaviour of this expandable template.

  • [expandableRows]="true or false" => Determine whether to show/hide the expandable rows section.
  • [expandOnRowClick]="true or false" => Tells data table to expand the table row on user row click event when expandable rows feature is enabled.
  • [showRowExpandLoadingSpinner]="true or false" => Enables loading spinner when data is fetching to the table row expandable section.
