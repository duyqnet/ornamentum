Data Table Paging

  • This help users to show pagination options in the data table.
  • To enable pagination it is required to set the pageable property into true.
  • If the value if not specified for the above property pagination will not be shown to the user.
  • When the pagination is enabled users can specify the number of items to be shown in a specific page, by using the limit text box.
  • Users can use the navigation arrows in the pagination bar to navigate to the other pages if there are more pages.
