Column Binding

Column binding section is consists of set of basic properties which will describes how we can use them for data table column data binding and manipulate data inside the column. The grid can facilitate a column for every item of the data array.

What You Will See

The following code snippet will depict the basic usage of column data binding. For use all it's associated properties it is mandatory to define field property, otherwise functionality may differ.

Useful Properties

Field

field: string  
Default Value: undefined
Target Component: <ng-data-table-column>

Field property is a mandatory property that bound to a column. This property will act as a identifier for access specific data set in your data array. The field name should be a valid Javascript identifier and should only contains Alphanumeric Characters.

Title

title: string  
Default Value: undefined
Target Component: <ng-data-table-column>

Useful to set a name for the field that column is bounded. If the property is not provided ornamentum will render column title as blank.

CSS Class

cssClass: string  
Default Value: undefined
Target Component: <ng-data-table-column>

This property is useful if you want to inject a CSS class into your data table column cells. you can provide a string or a function which returns a CSS class or classes which space delimiter separated without dot selector.

Width

width: number | string  
Default Value: undefined
Target Component: <ng-data-table-column>

The width can be used to set a static width for grid column. It accepts number which automatically converts it's unit to pixels or you can manually define a string with respective CSS units (px, cm, mm etc).

Visible

visible: boolean  
Default Value: true
Target Component: <ng-data-table-column>

This property is useful if you want to show your column in the data table. If it is set to false, ornamentum will not renders the column and it's content but include it in Show Column Selector dropdown as not selected.

Show In Column Selector

showInColumnSelector: boolean  
Default Value: true
Target Component: <ng-data-table-column>

If this property is set to true, the column will be displayed on Show Column Selector dropdown with field title and selected state, By enabling this functionality end users able to customize what they want to see.
