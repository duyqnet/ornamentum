Loading Spinner Template

Loading Spinner Template, provides an option to customize the appearance of data loading spinner as desired that is displayed while data loading or fetching. To define the loading spinner template, nest an <ng-template> tag with the showLoadingSpinner directive inside.
Further information can be found upon this topic in Data Loading Spinner section.
