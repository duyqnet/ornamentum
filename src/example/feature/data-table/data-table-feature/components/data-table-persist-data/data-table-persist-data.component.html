Data Table Data Persistence

  • Data enables users to persist data they are doing state transitions. As an example navigating through the routes.
  • So in here when users do search results, filtering, sorting etc..
  • The updated state of the data table will be stored in the users's browser storage.
  • To enable data persistence users have to set the persistTableState property in to true. The default value for this config is false.
  • There are two types of storage modes can be configured when using this data table. Those are,
    • Local Storage
    • Session Storage
    In here, we are using session storage as the default storage mode. The following configurations shows what are the other things we need to use when persisting data.
    • [storageMode]="DataTableStorageMode.SESSION" or "DataTableStorageMode.LOCAL"
    • [id]="state saving key name in the browser storage"
